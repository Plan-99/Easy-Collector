이 가이드는 Easy Trainer의 **9단계 워크플로우**를 처음부터 끝까지 한 번 경험하는 워크스루입니다.
**Tutorial Mode**(시뮬레이터 기반)로 진행되므로 실제 로봇 없이도 모든 단계를 그대로 따라할 수 있습니다.

<Callout type="tip">
완주에 약 30–60분 정도 걸립니다 (학습 시간 포함). GPU가 있으면 더 짧습니다.
</Callout>

## 1. 튜토리얼 모드 켜기

먼저 시뮬레이터를 부팅하고 화면 가이드를 켭니다.

<Steps>
1. **사이드바 하단의 Tutorial Mode 토글을 켭니다.**

   토글 옆에 "Switching…" → "Sim running" 순서로 상태가 바뀝니다.

   <DocsImage
     src="/docs/quickstart/01-tutorial-toggle.png"
     alt="MainLayout 사이드바의 Tutorial Mode 토글"
     status="missing"
   />

2. **(선택) Pipeline Guide로 전체 흐름을 미리 봅니다.**

   사이드바의 **Pipeline Guide** 버튼을 누르면 Sensors → Robots → Assembly → Workspace → Training → Inference 6단계 설명을 볼 수 있습니다.

   <DocsImage
     src="/docs/quickstart/03-pipeline-guide.png"
     alt="PipelineGuideDialog 6단계 stepper"
     status="missing"
   />
</Steps>

## 6. 워크스페이스 만들기

워크스페이스는 **한 작업(task) 단위로 센서/로봇/데이터셋을 묶어둔 컨테이너**입니다.

<Steps>
1. **사이드바의 Workspace로 이동하고 셀렉트에서 Create New Workspace +를 선택합니다.**

   <DocsImage
     src="/docs/quickstart/21-workspace-create.png"
     alt="Workspace 셀렉트 dropdown — Create New Workspace +"
     status="missing"
   />

2. **이름을 입력하고 Create를 누릅니다.**

   <DocsImage
     src="/docs/quickstart/22-workspace-form.png"
     alt="Workspace 생성 폼"
     status="missing"
   />

3. **setting 탭에서 Sensor Setting을 펼치고 + Add Sensor로 시뮬 카메라를 추가합니다.**

   <DocsImage
     src="/docs/quickstart/23-workspace-sensor.png"
     alt="Sensor Setting expansion — 센서 추가 직후"
     status="missing"
   />

4. **추가한 센서를 클릭하면 우측 본문에 crop / rotation 패널이 열립니다.**

   드래그로 crop 영역을 설정하거나, **Cropped Area**의 x1/y1/x2/y2를 직접 입력할 수 있습니다. **Rotation**으로 90/180/270도 회전도 가능합니다.

   <DocsImage
     src="/docs/quickstart/24-workspace-sensor-crop.png"
     alt="센서 config 패널 — crop 캔버스"
     status="missing"
   />

5. **Robot Setting을 펼치고 4단계에서 만든 Assembly를 선택합니다.**

   <DocsImage
     src="/docs/quickstart/25-workspace-robot.png"
     alt="Robot Setting expansion — Assembly 선택"
     status="missing"
   />

6. **로봇을 선택하면 Home Pose 패널이 열립니다.**

   먼저 RobotPendant 등으로 원하는 자세를 만든 뒤, Home Pose 섹션의 **sync** 버튼으로 현재 자세를 가져오고 **save**로 저장합니다. 데이터 수집 시작 시 로봇이 항상 이 자세로 먼저 돌아갑니다.

   <DocsImage
     src="/docs/quickstart/26-workspace-homepose.png"
     alt="Home Pose 입력 + sync/save 버튼"
     status="missing"
   />

7. **Task Setting을 펼치고 Episode Length를 설정합니다.**

   한 에피소드의 최대 프레임 수입니다. 기본값(예: 100)으로 시작해도 됩니다.

   <DocsImage
     src="/docs/quickstart/27-workspace-task.png"
     alt="Task Setting — Episode Length"
     status="missing"
   />

8. **data 탭으로 가서 + Add Dataset Folder로 데이터셋을 만듭니다.**

   <DocsImage
     src="/docs/quickstart/28-workspace-dataset-add.png"
     alt="Add Dataset Folder 폼"
     status="missing"
   />
</Steps>

## 7. 데이터 수집 (10 에피소드)

<Steps>
1. **우측 MonitoringWindow에 센서 피드와 로봇 상태가 모두 보이는지 확인합니다.**

   상단에 카메라 뷰, 그 아래에 각 관절의 현재값/목표값이 화살표로 표시됩니다.

   <DocsImage
     src="/docs/quickstart/29-monitoring-overview.png"
     alt="MonitoringWindow — 센서 피드 + 로봇 상태"
     status="missing"
   />

2. **하단 셋업 행을 설정하고 REC 버튼 위 🏠 뱃지를 켭니다.**

   - **Dataset**: 6단계에서 만든 데이터셋 선택
   - **Language Instruction**: 작업 지시 (예: "그릇을 집어서 옮긴다")
   - **Teleop Type**: `keyboard`
   - **🏠 뱃지**: 파란색(ON)으로 켭니다. 회색(OFF)이면 클릭해서 활성화. ON이면 매 에피소드 시작 시 워크스페이스의 [Home Pose](/docs/workspace/home-pose)로 자동 이동합니다.

   <DocsImage
     src="/docs/quickstart/30-record-setup.png"
     alt="MonitoringWindow 수집 setup row"
     status="missing"
   />

3. **REC 버튼을 누르면 로봇이 Home Pose로 이동한 뒤 수집이 시작됩니다.**

   "Moving to Home Pose" 스피너가 떠 있는 동안에는 키 입력이 무시됩니다.

   <DocsImage
     src="/docs/quickstart/31-moving-home.png"
     alt="Moving to Home Pose 스피너"
     status="missing"
   />

   수집이 시작되면 progress bar와 SUCCESS / DONE / STOP 버튼이 나타납니다.

   - **SUCCESS (C)** — 동작이 잘 됐을 때 누름. 성공 라벨로 저장됩니다.
   - **DONE** — 한 에피소드를 완료하고 데이터셋에 저장합니다.
   - **STOP** — 에피소드를 버림. 데이터셋에 저장되지 않습니다.

   설정한 [Episode Length](/docs/workspace/episode-config)만큼 시간이 지나거나 **DONE**을 클릭하면 그 에피소드가 데이터셋에 저장됩니다. 에피소드 종료 후에는 다음 에피소드가 자동으로 시작되니 **같은 동작을 10번 반복**합니다.

4. **data 탭에서 데이터셋을 펼쳐 10개 에피소드가 쌓였는지 확인합니다.**

   <DocsImage
     src="/docs/quickstart/33-dataset-10-episodes.png"
     alt="data 탭 expansion — 10 에피소드"
     status="missing"
   />
</Steps>

<Callout type="tip">
같은 작업이라도 **시작 위치/속도/각도**가 약간씩 달라야 학습이 잘 됩니다. 같은 패턴 10번보다 다양한 패턴 10번이 훨씬 좋습니다.
</Callout>

## 8. ACT로 학습하기

<Steps>
1. **사이드바에서 Train으로 이동합니다.**

   상단에서 같은 워크스페이스를 선택하면 stepper가 나타납니다. **Step 1: Select Datasets**가 활성화됩니다.

     alt="TrainPage stepper Step 1"
     status="missing"
   />

2. **방금 만든 데이터셋 카드를 선택하고 Continue를 누릅니다.**

   카드 우상단의 체크박스가 켜지고 테두리가 파란색으로 바뀝니다.

   <DocsImage
     src="/docs/quickstart/36-train-dataset-select.png"
     alt="Step 1 — 데이터셋 선택"

   <DocsImage

     alt="Step 2 — Create New Policy"
     status="missing"
   />

4. **Policy Type을 ACT로 선택하면 ACT 파라미터 폼이 표시됩니다.**

   클릭하면 다이얼로그를 다시 열어 최종 loss와 best epoch을 확인할 수 있습니다.

   <DocsImage
     src="/docs/quickstart/42-train-done.png"
     alt="학습 완료 — 우하단 floating 버튼"
     status="missing"
   />
</Steps>

<Callout type="info">
ACT 기본 파라미터로 10 에피소드 학습은 보통 **수 분~10여 분** 정도 걸립니다. GPU 성능과 VRAM에 따라 차이가 납니다.
</Callout>

## 9. 추론 실행

<Steps>
1. **Workspace로 돌아가 inference 탭을 엽니다.**

   방금 만든 체크포인트가 카드로 보입니다.

   <DocsImage
     src="/docs/quickstart/43-inference-list.png"
     alt="Workspace inference 탭 — 체크포인트 카드"
     status="missing"
   />

2. **체크포인트 카드를 클릭하면 우측 MonitoringWindow에 inference 헤더가 나타납니다.**

   - **Hz**: 추론 주기 (예: 30)
   - **🏠 배지**: 추론 시작 시 Home Pose로 먼저 이동할지 (파란색=ON, 회색=OFF)

   <DocsImage
     src="/docs/quickstart/44-inference-setup.png"
     alt="MonitoringWindow inference 헤더"
     status="missing"
   />

3. **Start Inference를 누르면 정책이 실시간으로 액션을 추론하기 시작합니다.**

   - **Succeed** (녹색이면 > 0.7) — 학습된 성공 분포와의 유사도
   - **OOD** (주황 > 1.0이면 빨강) — 분포 이탈 정도

   동작이 마음에 들지 않거나 위험해 보이면 **Stop**을 눌러 즉시 멈춥니다.

   <DocsImage
     src="/docs/quickstart/45-inference-running.png"
     alt="추론 중 — Succeed / OOD 배지"
     status="missing"
   />
</Steps>

## 10. 플래너로 묶어 실행하기

마지막으로, 방금 만든 체크포인트를 [플래너](/docs/planner)에 넣어 **시작 자세 이동 → 추론 → 대기**를 한 번에 실행해 봅니다. 정책이 끝나는 시점이 매번 다른 작업도 한 줄짜리 플랜으로 자동화할 수 있습니다.

<Steps>
1. **사이드바에서 Planner로 이동하고 상단 드롭다운 → + Create New Planner를 누릅니다.**

   이름을 입력하고 CREATE를 누르면 빈 플래너가 생성됩니다.

   <DocsImage
     src="/docs/planner/create-02-new.png"
     alt="새 플래너 만들기 다이얼로그"
   />

2. **좌측 패널의 + ADD WORKSPACES 버튼으로 6단계에서 만든 워크스페이스를 추가합니다.**

   <DocsImage
     src="/docs/planner/workspaces-02-multiselect.png"
     alt="워크스페이스 다중 선택 다이얼로그"
   />

3. **가운데 패널의 새 블록 버튼으로 블록을 3개 추가합니다.**

   - **Joint Position** — 시작 자세로 이동 (워크스페이스 선택 후 펜던트로 자세를 만들고 **현재 자세 적용**으로 저장)
   - **Checkpoint** — 9단계에서 만든 체크포인트 선택 + 시간(예: 10초) + Hz 입력
   - **Time Sleep** — 1–2초 대기

   <DocsImage
     src="/docs/planner/blocks-list-01-row.png"
     alt="블록 목록 — Joint Position / Checkpoint / Time Sleep 3개 행"
   />

4. **가운데 패널 상단의 전체 실행(녹색 play) 버튼을 누릅니다.**

   사전 검증을 통과하면 첫 블록부터 차례대로 실행됩니다. 이상하면 곧바로 **중지**를 누르세요.

   <DocsImage
     src="/docs/planner/run.png"
     alt="플랜 실행 중 — 좌측 진행 상태 + 우측 모니터링 윈도우"
   />
</Steps>

자세한 사용법은 [플래너 개요](/docs/planner) → [플래너 생성 / 워크스페이스 추가](/docs/planner/create-and-workspaces) → [블록 추가 / 플랜 실행](/docs/planner/blocks-and-run) 순서로 참고하세요.
