플래너는 좌측 메뉴의 **Planner** 페이지에서 사용합니다. 학습된 정책 한 개만으로 끝낼 수 없는 긴 작업을, 여러 **블록**(관절 이동 · 체크포인트 실행 · 시간 대기)을 순서대로 엮어 자동으로 실행합니다.

## 사전 조건

플래너를 사용하려면 다음이 미리 준비되어 있어야 합니다.

- 하나 이상의 [워크스페이스](/docs/workspace)
- 워크스페이스에 연결된 [어셈블리와 로봇](/docs/teleoperation/assembly)
- 체크포인트 블록을 사용한다면 [학습이 끝난 체크포인트](/docs/training)

<Callout type="info">
플래너 실행은 워크스페이스의 로봇이 모두 켜진 상태에서만 시작할 수 있습니다. 시작 전에 좌측 워크스페이스 패널에서 각 로봇의 상태가 **켜짐**인지 확인하세요.
</Callout>

## 페이지 구성

플래너 페이지는 좌·중·우 세 영역으로 나뉘어 있습니다.

- **좌측 패널** — 이 플래너가 사용할 워크스페이스 목록. 각 워크스페이스 아래에 센서/로봇 토글이 있습니다.
- **가운데 패널** — 블록 목록과 **전체 실행** / **중지** 버튼.
- **우측 패널** — 모니터링 윈도우. 센서 영상과 로봇 상태를 실시간으로 보여줍니다.

## 블록 종류

| 종류 | 의미 |
|------|------|
| **Joint Position** | 지정한 워크스페이스의 로봇을 저장된 관절 자세로 이동시킵니다. |
| **Checkpoint** | 지정한 체크포인트로 실시간 추론을 일정 시간 동안 실행합니다. |
| **Time Sleep** | 지정한 초만큼 대기합니다. |

## 진행 순서

1. [플래너 만들기](/docs/planner/create) — 새 플래너를 생성하고 선택합니다.
2. [워크스페이스 추가](/docs/planner/workspaces) — 이 플래너에서 사용할 워크스페이스를 고릅니다.
3. [블록 구성](/docs/planner/blocks) — 실행할 블록을 추가하고 순서를 정합니다.
4. [전체 실행](/docs/planner/run) — 플랜을 한 번에 실행하고 진행 상황을 모니터링합니다.

## 다음 단계

플래너로 여러 체크포인트를 엮어 동작이 끊이지 않게 만들었다면, 부족한 단계를 발견할 때마다 [데이터 수집](/docs/data-collection)으로 돌아가 더 많은 시연을 모으거나 [데이터 증강](/docs/datasets/augment)으로 학습 샘플을 늘릴 수 있습니다.
