플래너의 좌측 패널에서 이 플래너가 사용할 워크스페이스를 추가합니다. 같은 플래너 안에 여러 워크스페이스를 둘 수 있고, 각 블록은 그 중 하나를 골라 동작합니다.

<Steps>
1. **좌측 패널 상단의 워크스페이스 추가 버튼을 누릅니다.**

   <DocsImage
     src="/docs/planner/workspaces-01-add-button.png"
     alt="워크스페이스 추가 버튼"
     status="missing"
   />

2. **워크스페이스 선택 다이얼로그에서 사용할 항목을 체크합니다.**

   여러 개를 동시에 선택할 수 있습니다. 저장하면 좌측 패널에 항목이 추가됩니다.

   <DocsImage
     src="/docs/planner/workspaces-02-multiselect.png"
     alt="워크스페이스 다중 선택 다이얼로그"
     status="missing"
   />

3. **추가된 워크스페이스를 펼쳐 센서와 로봇 상태를 확인합니다.**

   카드 헤더를 클릭하면 아래로 펼쳐집니다.

   - **센서** 영역 — 일반 센서는 토글로 켜고 끄며, 커스텀 센서는 **TOPIC ON/OFF**로 외부 토픽 존재 여부가 표시됩니다.
   - **로봇** 영역 — 어셈블리에 연결된 로봇 목록. 일반 로봇은 토글, 커스텀 로봇은 **TOPIC ON/OFF**로 표시됩니다. 켜지지 않거나 오류가 나면 빨간 **에러** 배지가 보입니다.

   <DocsImage
     src="/docs/planner/workspaces-03-expanded.png"
     alt="워크스페이스 펼친 모습 — 센서/로봇 토글"
     status="missing"
   />

4. **플래너에서 빼고 싶은 워크스페이스는 헤더 우측의 X 버튼으로 제거합니다.**

   제거해도 워크스페이스 자체가 삭제되지는 않으며, 이 플래너에서만 빠집니다.

   <DocsImage
     src="/docs/planner/workspaces-04-remove.png"
     alt="워크스페이스 제거 X 버튼"
     status="missing"
   />
</Steps>

<Callout type="warning">
플래너 실행 전에는 사용 중인 워크스페이스의 모든 로봇이 켜져 있어야 합니다. 로봇이 꺼진 상태에서 실행을 시도하면 사전 검증에서 실패하고 어떤 로봇이 켜져 있지 않은지 알림으로 표시됩니다.
</Callout>
