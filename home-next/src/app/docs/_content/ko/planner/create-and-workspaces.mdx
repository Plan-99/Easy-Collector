플래너는 워크스페이스나 어셈블리와 마찬가지로 이름이 있는 저장 단위입니다. 한 플래너는 사용할 워크스페이스 목록과 그룹별 블록 순서(`plans`)를 함께 저장합니다. 이 문서에서는 플래너를 새로 만들고, 그 안에 사용할 워크스페이스를 추가하는 흐름을 다룹니다.

## 플래너 만들기

<Steps>
1. **상단의 플래너 선택 드롭다운을 엽니다.**

   드롭다운을 펼치면 기존 플래너 목록과 맨 아래 **+ Create New Planner** 항목이 보입니다.

   <DocsImage
     src="/docs/planner/create-01-select.png"
     alt="플래너 선택 드롭다운"
   />

2. **드롭다운 안의 + Create New Planner 항목을 클릭하고 이름을 입력합니다.**

   다이얼로그에서 이름을 입력하고 **CREATE**를 누르면 새 플래너가 생성되어 자동으로 선택됩니다.

   <DocsImage
     src="/docs/planner/create-02-new.png"
     alt="새 플래너 만들기 다이얼로그"
   />

3. **기존 플래너를 편집하거나 삭제하려면 드롭다운 항목 우측의 아이콘을 사용합니다.**

   - 연필 아이콘 — 이름 변경
   - 휴지통 아이콘 — 삭제. 화면 하단에 확인 알림이 뜨면 **delete**를 눌러야 실제로 삭제됩니다.

   <DocsImage
     src="/docs/planner/create-04-edit-delete.png"
     alt="플래너 항목의 편집(연필)/삭제(휴지통) 아이콘"
   />
</Steps>

플래너를 선택하면 가운데와 우측 패널이 활성화되고 워크스페이스/블록을 편집할 수 있습니다.

## 워크스페이스 추가하기

플래너의 좌측 패널에서 이 플래너가 사용할 워크스페이스를 추가합니다. 같은 플래너 안에 여러 워크스페이스를 둘 수 있고, 각 블록은 그 중 하나를 골라 동작합니다.

<Steps>
1. **좌측 패널 상단의 + ADD WORKSPACES 버튼을 누릅니다.**

   <DocsImage
     src="/docs/planner/workspaces-01-add-button.png"
     alt="워크스페이스 추가 버튼"
   />

2. **워크스페이스 선택 다이얼로그에서 사용할 항목을 체크합니다.**

   여러 개를 동시에 선택할 수 있습니다. **SAVE**를 누르면 좌측 패널에 항목이 추가됩니다.

   <DocsImage
     src="/docs/planner/workspaces-02-multiselect.png"
     alt="워크스페이스 다중 선택 다이얼로그"
   />

3. **추가된 워크스페이스의 센서와 로봇 상태를 우측 모니터링 윈도우에서 확인합니다.**

   추가가 끝나면 우측 모니터링 윈도우에 해당 워크스페이스의 카메라 미리보기와 로봇 관절 상태가 표시됩니다. 좌측 패널의 카드를 클릭하면 카드 안에서도 센서/로봇 토글을 직접 켜고 끌 수 있습니다.

   - **센서** — 일반 센서는 토글, 커스텀 센서는 **TOPIC ON/OFF** 표시.
   - **로봇** — 어셈블리에 연결된 로봇 목록. 일반 로봇은 토글, 커스텀 로봇은 **TOPIC ON/OFF**. 오류 시 빨간 **에러** 배지가 보입니다.

   <DocsImage
     src="/docs/planner/workspaces-03-expanded.png"
     alt="추가된 워크스페이스의 센서/로봇 상태가 모니터링 윈도우에 표시된 모습"
   />

4. **플래너에서 빼고 싶은 워크스페이스는 좌측 패널 항목 우측의 X 버튼으로 제거합니다.**

   제거해도 워크스페이스 자체가 삭제되지는 않으며, 이 플래너에서만 빠집니다.

   <DocsImage
     src="/docs/planner/workspaces-04-remove.png"
     alt="좌측 패널의 워크스페이스 항목과 우측 X 제거 버튼"
   />
</Steps>

## 그룹은 자동으로 만들어집니다

워크스페이스를 추가하면 백엔드가 어셈블리에 포함된 로봇 ID를 비교해서 **그룹**을 자동으로 계산합니다.

- 로봇을 **하나라도 공유하는** 워크스페이스끼리는 같은 그룹으로 묶입니다.
- 공유 로봇이 없으면 서로 다른 그룹이 되고, 우측 Plans 타임라인에 트랙이 한 줄씩 추가됩니다.
- 워크스페이스를 추가하거나 빼면 그룹이 다시 계산됩니다. 기존 블록은 가능한 한 원래 그룹에 그대로 남고, 더 이상 어디에도 속하지 않는 블록만 자동으로 정리됩니다.

같은 그룹의 블록은 순차로, 다른 그룹은 병렬로 실행되며, 그룹 간 동기화는 [Sync 블록](/docs/planner/blocks-and-run#sync-블록)으로 맞춥니다.

<Callout type="warning">
플래너 실행 전에는 사용 중인 워크스페이스의 모든 로봇이 켜져 있어야 합니다. 로봇이 꺼진 상태에서 실행을 시도하면 사전 검증에서 실패하고 어떤 로봇이 켜져 있지 않은지 알림으로 표시됩니다.
</Callout>
