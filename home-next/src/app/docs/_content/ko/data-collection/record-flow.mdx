데이터 수집은 다음 4단계로 진행됩니다.

<Steps>
1. **REC 버튼을 누릅니다.**

   <DocsImage
     src="/docs/data-collection/record-01-rec.png"
     alt="REC 버튼 hover"
   />

2. **(Home Pose ON일 때) 로봇이 Home Pose로 이동 → 수집이 시작되어 progress bar + SUCCESS / DONE / STOP 버튼이 나타나고, Episode Length 도달 또는 DONE 버튼으로 한 에피소드를 마무리합니다.**

   "Moving to Home Pose" 스피너가 표시되는 동안에는 키보드/리더 입력이 무시됩니다. 이후 사람이 텔레오퍼레이션으로 작업을 시연하면 매 프레임이 자동 저장됩니다.

   <DocsImage
     src="/docs/data-collection/record-02-moving.png"
     alt="Moving to Home Pose 스피너 → 수집 중 progress bar + SUCCESS / DONE / STOP 버튼 → progress 100% 직전"
   />
</Steps>

DONE 후 **Home Pose ON**이면 다시 2단계로 돌아가 다음 에피소드가 자동 시작됩니다. OFF면 곧바로 3단계로 진입합니다.

[Stop / Done / Success 버튼의 차이](/docs/data-collection/buttons) 페이지에서 각 버튼을 어떤 상황에 사용해야 하는지 확인하세요.
