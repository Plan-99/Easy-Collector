리플레이는 수집된 에피소드의 액션을 실제 로봇에 다시 흘려보내는 기능입니다. 결과를 눈으로 확인하거나, 다른 액션 형식으로 다시 캡처해 새 데이터셋을 만들 수 있습니다.

리플레이는 **워크스페이스 → data 탭**에서 시작합니다. (전역 데이터셋 탭에서는 리플레이가 아닌 [편집/증강/에피소드 편집](/docs/datasets) 작업을 합니다.)

<Steps>
1. **워크스페이스의 data 탭에서 데이터셋을 펼치고 에피소드를 클릭합니다.**

   <DocsImage
     src="/docs/data-collection/replay-01-select.png"
     alt="에피소드 선택"
     status="missing"
   />

2. **MonitoringWindow에 replay 헤더가 나타나면 Replay Action Type을 고르고, 필요하면 Capture Episode를 켭니다.**

   - **qaction** — 저장된 관절 명령을 그대로 재생
   - **ee_delta_action** — EE 델타로 변환해 재생 (워크스페이스의 IK 설정과 결합 가능성 검증에 유용)

   Capture Episode 체크박스를 켜고 Target Dataset을 지정하면 재생되는 동작이 새 데이터셋에 다시 저장됩니다. 모든 센서가 ON이어야 활성화됩니다.

   <DocsImage
     src="/docs/data-collection/replay-02-replay-panel.png"
     alt="Replay 헤더 — Action Type(qaction/ee_delta_action) + Capture Episode + Target Dataset"
     status="missing"
   />

3. **Hz를 입력하고 Play를 누릅니다.**

   캡처 모드일 때는 진행률 progress bar가 표시됩니다.

   <DocsImage
     src="/docs/data-collection/replay-03-progress.png"
     alt="Capturing X% progress"
     status="missing"
   />
</Steps>

<Callout type="warning">
리플레이 중에는 로봇이 실제로 움직입니다. 주변에 사람이나 물체가 없는지 확인하고 시작하세요.
</Callout>
