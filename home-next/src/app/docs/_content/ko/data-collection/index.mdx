데이터 수집은 워크스페이스의 **MonitoringWindow**(우측 패널)에서 진행됩니다.
사람이 텔레오퍼레이션으로 작업을 시연하면 매 프레임 (이미지 + 관절 상태 + 액션)이 데이터셋에 저장됩니다.

## 사전 조건

데이터 수집을 시작하려면 워크스페이스에 다음이 모두 설정되어 있어야 합니다.

- ✅ [Home Pose](/docs/workspace/home-pose) — 매 에피소드 시작 자세
- ✅ [Episode Length](/docs/workspace/episode-config) — 한 에피소드의 최대 프레임 수
- ✅ [데이터셋 폴더](/docs/workspace/dataset-folder) — 결과 저장 위치
- ✅ [텔레오퍼레이션 테스트](/docs/teleoperation/testing) — 입력 모드가 실제로 동작하는지 확인

## 몇 개의 에피소드를 모아야 하나

| 작업 복잡도 | 권장 에피소드 수 |
|----------|---------------|
| 단순 pick-and-place (고정 위치) | 10–20 |
| 단순 pick-and-place (위치 가변) | 30–50 |
| 양손 협조, 정밀 조작 | 100+ |
| 여러 객체 / 단계가 있는 작업 | 200+ |

같은 작업이라도 **시작 위치 / 속도 / 각도**가 약간씩 달라야 학습이 잘 됩니다.

## 다음 단계

충분한 에피소드를 모았다면 [학습](/docs/training)으로 넘어가 정책을 학습시키세요. 데이터를 늘리고 싶다면 [데이터 증강](/docs/datasets/augment)으로 시연 한 번에서 여러 학습 샘플을 만들 수 있습니다.
