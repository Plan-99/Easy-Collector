데이터 수집은 워크스페이스의 **MonitoringWindow**(우측 패널)에서 진행됩니다.
사람이 텔레오퍼레이션으로 작업을 시연하면 매 프레임 (이미지 + 관절 상태 + 액션)이 데이터셋에 저장됩니다.

## 한눈에 보기

1. [텔레오퍼레이션 모드 선택](/docs/data-collection/teleop-mode) — 키보드 / 리더 로봇 / VIVE / Motion Planning
2. [Home Pose 귀환 여부](/docs/data-collection/home-pose-return) — 매 에피소드 시작 시 Home Pose로 갈지
3. [수집 흐름](/docs/data-collection/record-flow) — REC → Moving → 수집 → DONE
4. [Stop / Done / Success 버튼](/docs/data-collection/buttons) — 어떤 상황에 어떤 버튼을 누르는가

<Callout type="info">
워크스페이스에 [Home Pose](/docs/workspace/home-pose), [Episode Length](/docs/workspace/episode-config), [데이터셋 폴더](/docs/workspace/dataset-folder)가 설정되어 있어야 합니다.
</Callout>
