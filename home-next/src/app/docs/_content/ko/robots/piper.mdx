Piper는 CAN 버스로 직접 통신하는 6-DoF 협동 로봇입니다. **별도의 vendor 티치펜던트가 없고** USB-CAN 어댑터로 PC에 직접 연결하는 구조라 하드웨어 케이블 점검이 핵심입니다. 등록 절차 자체는 [공통 절차](/docs/robots#공통-절차)와 동일합니다.

## 사전 조건

- [모듈 추가](/docs/modules)에서 **Piper** 모듈 설치
- USB-CAN 어댑터를 PC와 Piper 사이에 연결
- Piper 본체에 전원 인가, 부팅 완료
- 호스트에서 `can0` 인터페이스가 보이는지 확인:

  ```bash
  ip link show can0
  ```

  `can0`이 없거나 `DOWN`이면 USB 케이블 / 어댑터 / 드라이버를 점검합니다.

## 하드웨어 연결

USB-CAN 어댑터의 D-sub / 단자가 Piper 베이스의 CAN 포트에 연결되어 있어야 합니다. 잘못 연결하면 토글 ON 시 ONLINE이 되지 않거나 LOADING 상태에서 멈춥니다.

## 등록 입력값

[공통 절차](/docs/robots#공통-절차) Step 2에서 다음 값을 입력합니다.

- **Robot Type**: `piper`
- **CAN Port**: `can0` (기본값으로 두면 대부분 OK)

이후 하단의 조작 버튼을 통해 로봇의 움직임을 검증합니다 (공통 절차 Step 4).

<Callout type="warning">
처음 토글을 켤 때 작은 진동과 함께 토크가 ON됩니다. 사람이 닿지 않는 위치에서 시작하세요.
</Callout>
