Easy Trainer는 [모듈 추가](/docs/modules)에서 설치한 로봇 드라이버를 운영자 UI에서 인스턴스 단위로 등록해 사용합니다.
같은 모델 로봇이 여러 대 있다면 **각각 별도로 등록**합니다 (예: `piper_left`, `piper_right`).

## 두 가지 등록 방식

| 방식 | 언제 |
|------|------|
| **모듈로 추가** | 등록된 vendor가 모듈로 제공되는 경우 (Piper, Dynamixel, Unitree, Jaka, Fairino, OMX, RBPodo, Kinova, Techman 등) |
| **Custom Robot** | 자체 개발 로봇이거나 vendor가 ROS 2 인터페이스만 제공 |

상세 절차:

- [Piper](/docs/robots/piper) — CAN 기반 6-DoF
- [Fairino](/docs/robots/fairino) — IP 기반 협동 로봇 (FR5 등)
- [ROBOTIS OMX](/docs/robots/omx) — Dynamixel USB 시리얼 기반 5-DoF (OpenManipulator-X와 다른 제품)
- [ROBOTIS OMY](/docs/robots/omy) — DYNAMIXEL-Y 기반 6-DoF, ROS 2 네이티브 (OMY-3M / OMY-F3M)
- [Custom Robot](/docs/robots/custom) — ROS 2 토픽 / 서비스 / 액션 기반

## 공통 절차

<Steps>
1. **Robots → Management에서 Add Robot.**

   <DocsImage
     src="/docs/robots/add-robot-button.png"
     alt="RobotPage — Add Robot 버튼"
     status="missing"
   />

2. **Robot Type을 선택하고 연결 정보(CAN/IP/Serial 등)를 입력.**

   <DocsImage
     src="/docs/robots/robot-form.png"
     alt="RobotForm — Robot Type 선택 + 연결 정보 입력"
     status="missing"
   />

3. **저장 후 카드의 토글을 켜서 ONLINE 상태를 확인.**

   <DocsImage
     src="/docs/robots/robot-online.png"
     alt="로봇 카드 — 토글 ON, 상태 ONLINE"
     status="missing"
   />

4. **하단의 조작 버튼을 통해 로봇의 움직임을 검증.**
5. **(선택) 자주 쓰는 자세를 RobotPendant 패널 상단의 ADD POSE로 저장합니다.**

   저장된 pose 버튼을 클릭하면 즉시 그 자세로 이동, 우클릭으로 삭제할 수 있습니다. Home Pose 외에도 작업 시작·종료 자세, 안전 자세 등을 미리 등록해 두면 텔레오퍼레이션과 데이터 수집이 편해집니다.

   <DocsImage
     src="/docs/robots/add-pose-button.png"
     alt="RobotPendant 패널 — ADD POSE 버튼 + 저장된 pose 버튼들"
     status="missing"
   />
</Steps>

<Callout type="info">
이 페이지의 **RobotPendant**는 운영자 UI(브라우저)의 하단 패널입니다. 일부 vendor가 컨트롤박스와 함께 제공하는 **티치펜던트(teach pendant)**와는 별개입니다. vendor 티치펜던트에서만 가능한 통신/IP 설정은 [Piper](/docs/robots/piper) / [Fairino](/docs/robots/fairino) 등 vendor별 페이지를 참고하세요.
</Callout>

## 다음 단계

로봇이 등록되면 [Assembly 구성](/docs/assembly)에서 좌·우 팔과 그리퍼를 한 묶음(=하나의 가상 로봇)으로 정의하세요. Assembly가 있어야 텔레오퍼레이션·워크스페이스 단계로 넘어갈 수 있습니다.

<Callout type="warning">
로봇 토글을 처음 켤 때 vendor에 따라 작은 진동과 함께 토크가 ON됩니다. 사람이나 물체가 닿지 않는 위치에서 시작하세요.
</Callout>
