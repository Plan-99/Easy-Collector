Fairino는 IP 네트워크로 통신하는 6-DoF 협동 로봇입니다 (FR5 등). 컨트롤박스와 함께 제공되는 **vendor 티치펜던트**에서 네트워크와 통신 모드를 먼저 설정해야 EasyTrainer가 SDK로 접속할 수 있습니다. 등록 절차 자체는 [공통 절차](/docs/robots#공통-절차)와 동일합니다.

## 사전 조건

- [모듈 추가](/docs/modules)에서 **Fairino** 모듈 설치
- 컨트롤박스가 PC와 같은 네트워크에 연결되어 IP가 할당되어 있음
- vendor 티치펜던트에서 외부 SDK 통신이 허용된 상태

## 티치펜던트에서 IP 확인

컨트롤박스의 LAN 포트와 PC를 같은 LAN에 연결한 뒤, 티치펜던트의 **설정 → 네트워크** 메뉴에서 컨트롤박스 IP를 확인합니다 (출고 기본값 예: `192.168.58.2`). PC 쪽 IP가 같은 서브넷에 있도록 맞춰주세요.

<Callout type="info">
Fairino 컨트롤박스는 LAN 포트가 두 개(`Eth0` / `Eth1`)인데, **외부 SDK 통신은 `Eth1`을 사용**합니다. PC와 `Eth1`을 연결하고 티치펜던트에서 확인한 **`Eth1`의 IP**를 EasyTrainer에 입력하세요. `Eth0` 쪽 IP를 입력하면 ping은 되더라도 SDK 접속이 실패할 수 있습니다.
</Callout>

확인 후 PC에서 `ping <controller-ip>`가 응답하면 LAN 연결 OK.

<Callout type="warning">
모델/펌웨어에 따라 티치펜던트에서 **외부 제어 모드(Remote / Auto)**를 활성화해야 SDK 명령이 통과합니다. 통신은 되지만 Enable 단계에서 멈춘다면 펜던트의 모드 설정을 다시 확인하세요. 정확한 메뉴 위치는 vendor 매뉴얼을 따르세요.
</Callout>

## 등록 입력값

[공통 절차](/docs/robots#공통-절차) Step 2에서 다음 값을 입력합니다.

- **Robot Type**: `fairino_fr5` (또는 사용 모델)
- **IP Address**: 티치펜던트에서 확인한 컨트롤박스 IP

이후 하단의 조작 버튼을 통해 로봇의 움직임을 검증합니다 (공통 절차 Step 4).

<Callout type="info">
Fairino SDK는 MoveIt(`ros-humble-moveit`)에 의존합니다. 모듈 설치 시 자동으로 설치되지만, 누락된 경우 모듈을 다시 설치하세요.
</Callout>
