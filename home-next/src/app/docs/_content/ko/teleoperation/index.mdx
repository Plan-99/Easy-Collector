텔레오퍼레이션은 사람이 로봇을 직접 조작해 시연(demonstration) 데이터를 만드는 과정입니다.
Easy Trainer는 4가지 입력 모드를 지원합니다.

## 모드 비교

| 모드 | 입력 장치 | 적합한 작업 | 학습 적합도 |
|------|----------|-----------|-----------|
| **Keyboard** | 키보드 | 정밀한 위치 조정, 디버깅 | 단순 동작에 ⭕ |
| **Leader Robot** | 마스터-슬레이브 팔 (Dynamixel) | 양손 협조 동작, 미세 조작 | 가장 자연스러움 ⭕⭕ |
| **VIVE** | HTC VIVE 컨트롤러 | 3D 공간 포즈 조작 | 중간 ⭕ |
| **Motion Planning** | 자동 경로 생성 | 반복 작업, 합성 데이터 | 보조용 |

## 5단계 진행 순서

1. [Assembly](/docs/teleoperation/assembly) — 좌/우 팔, 그리퍼를 한 단위로 묶기
2. [End-Effector 세팅](/docs/teleoperation/end-effector) — EE 오프셋 보정
3. [키보드](/docs/teleoperation/keyboard) — 키 매핑·step size 설정
4. [Leader Robot](/docs/teleoperation/leader-robot) — 리더 팔 캘리브레이션 (선택)
5. [테스트](/docs/teleoperation/testing) — 실제로 움직여 검증

## 처음 시작한다면

<Callout type="tip">
처음에는 **키보드만으로 시작**하는 것을 권장합니다. EE offset과 키 매핑 검증이 끝난 뒤 Leader Robot 캘리브레이션으로 넘어가도 늦지 않습니다.
</Callout>

## 다음 단계

텔레오퍼레이션 테스트가 끝났다면 [데이터 수집](/docs/data-collection)으로 시연을 시작하세요. 모든 모드는 [데이터 수집의 Teleop Type](/docs/data-collection/teleop-mode)에서 다시 선택할 수 있습니다.
