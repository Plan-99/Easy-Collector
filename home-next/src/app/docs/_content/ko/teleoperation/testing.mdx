설정이 끝났다면 실제로 움직여 봐야 안심입니다. 이 페이지는 데이터 수집 페이지로 가기 전 마지막 점검 단계입니다.

## 점검 절차

<Steps>
1. **Robots → Teleoperation에서 Assembly 탭을 엽니다.**

   하단 TeleopConsole이 활성화됩니다.

   <DocsImage
     src="/docs/teleoperation/testing-01-console.png"
     alt="TeleopConsole 초기 상태"
     status="missing"
   />

2. **Start Keyboard Teleop을 누른 뒤 콘솔 영역을 클릭해 포커스를 두고 매핑된 키를 누릅니다.**

   버튼이 주황색 "Stop Keyboard Teleop"으로 바뀌면 활성화 상태입니다. 왼쪽 키보드 로그에 키 이벤트가, 오른쪽 리더 로그(설정된 경우)에 리더 상태가 표시됩니다.

   <DocsImage
     src="/docs/teleoperation/testing-02-running.png"
     alt="Start Keyboard Teleop 활성 — 좌측 키보드 로그 + 우측 리더 로그"
   />

3. **(선택) Leader Robot이 설정되어 있다면 Start Leader Teleop도 눌러 보세요.**

   사람이 리더 팔을 움직이면 슬레이브가 같이 움직입니다.
</Steps>

<Callout type="info">
이 페이지에서 검증된 텔레오퍼레이션 모드만 [데이터 수집](/docs/data-collection)에서 안정적으로 사용할 수 있습니다.
</Callout>
