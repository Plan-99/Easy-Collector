수집한 에피소드는 워크스페이스의 **data 탭**에서 관리합니다.

## 가능한 작업

- [리플레이](/docs/datasets/replay) — 에피소드를 다시 재생, 액션 형식 비교, 다른 데이터셋으로 다시 캡처
- [증강](/docs/datasets/augment) — 색상·크롭 변형으로 데이터 양 늘리기
- [편집](/docs/datasets/edit) — 이름 변경, 병합, sensor/robot 매핑 재지정, 삭제

<Callout type="tip">
같은 작업이라도 데이터셋을 작업 단계별로 분리해 두면 학습 실험이 편합니다 (예: `pick_v1`, `pick_v2`, `pick_eval`).
</Callout>
