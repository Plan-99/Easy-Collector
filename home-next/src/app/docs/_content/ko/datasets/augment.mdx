증강은 같은 에피소드를 여러 변형으로 늘려 학습의 일반화 성능을 높이는 기법입니다.

## 어디서 하나

기본 진입점은 좌측 네비의 **데이터셋 탭**이고, 워크스페이스의 **data 탭**에서도 동일하게 가능합니다. 어느 쪽이든 데이터셋을 우클릭하면 메뉴가 나옵니다.

<Steps>
1. **데이터셋을 우클릭하고 Augment Dataset을 누릅니다.**

   <DocsImage
     src="/docs/datasets/context-menu.png"
     alt="데이터셋 우클릭 메뉴 — Edit / Augment / 다운샘플 / Merge / Export / Delete"
     status="missing"
   />

2. **Data Augmentation 다이얼로그가 풀스크린으로 열립니다.**

   변환 종류, 강도, 결과 데이터셋 이름 등을 설정합니다.

   <DocsImage
     src="/docs/datasets/augment-02-dialog.png"
     alt="DataAugmentationDialog 전체"
     status="missing"
   />
</Steps>

<Callout type="info">
증강은 새로운 에피소드 파일을 생성합니다. 원본은 그대로 유지되니 안심하고 실행하세요.
</Callout>
