증강(Augmentation)은 같은 에피소드를 여러 변형으로 늘려 학습의 일반화 성능을 높이는 기법입니다. 색상·노이즈·perspective 같은 시각 변환을 무작위로 섞어 한 에피소드에서 여러 학습 샘플을 만들어냅니다.

## 어디서 여나

좌측 네비의 **데이터셋 탭**에서 다음 중 하나로 [일괄 변경(Batch Edit)](/docs/datasets/batch-edit) 패널을 열고 **Augmentation** 섹션을 펼칩니다.

- 트리에서 **데이터셋 폴더 라벨**을 클릭
- 데이터셋 우클릭 → **Batch Edit**

## 사용법

`Augmentation` 섹션의 **Enable 토글**을 켠 뒤 다음 항목을 설정합니다.

- **Repeat** *(1~20)* — 에피소드당 만들 변형 개수. `N`을 입력하면 원본 1개당 무작위 변형 N개가 추가 생성됩니다.
- **Lightness / HSV / Disturbances / Salt & Pepper / Gaussian / Perspective** — 각 효과의 토글과 슬라이더로 강도를 조정합니다.

옵션을 마치면 패널 상단의 **출력 이름**(기본값 `<원본>_transformed`)을 확인하고 하단의 **Apply** 를 누릅니다. 원본 데이터셋은 그대로 유지되며, 새 데이터셋이 목록에 추가됩니다.

옵션별 세부 의미와 슬라이더 범위는 [일괄 변경 → Augmentation 섹션](/docs/datasets/batch-edit#augmentation-섹션)을 참고하세요.

<Callout type="tip">
증강 한 번에 다운샘플 / 크롭 / 회전을 같이 적용하려면 같은 패널에서 해당 섹션의 토글을 추가로 켜면 됩니다. 변환은 한 패스 안에서 **augment ×Repeat → downsample → crop → rotate** 순서로 처리됩니다.
</Callout>
