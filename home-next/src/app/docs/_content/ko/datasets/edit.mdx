데이터셋 단위로 이름을 바꾸거나, 여러 데이터셋을 합치거나, sensor/robot 매핑을 재지정하거나, 삭제하는 작업입니다.

## 어디서 하나

기본 진입점은 좌측 네비의 **데이터셋 탭**이고, 워크스페이스의 **data 탭**에서도 동일하게 가능합니다. 어느 쪽이든 데이터셋을 우클릭하면 **Edit / Augment / 다운샘플 / Merge / Export / Delete** 메뉴가 나옵니다.

## Edit

이름을 변경하거나 sensor/robot 매핑을 재지정합니다.

  src="/docs/datasets/edit-01-edit.png"

  status="missing"
/>

## Merge

여러 데이터셋을 하나로 합칩니다. 같은 작업의 다른 시점/사람의 시연을 묶을 때 유용합니다.

<DocsImage
  src="/docs/datasets/edit-02-merge.png"
  alt="Merge Dataset 폼"
  status="missing"
/>

## Sensor / Robot 매핑 재지정

워크스페이스의 sensor/robot 구성이 바뀌면 기존 데이터셋이 호환되지 않습니다. Edit 폼 안의 매핑 셀렉트로 옛 이름을 새 이름에 다시 연결할 수 있습니다.
