데이터셋 우클릭 메뉴에는 **Edit / Augment / Merge / Delete** 4개 항목이 있습니다.

## Edit

이름을 변경하거나 sensor/robot 매핑을 재지정합니다.

<DocsImage
  src="/docs/datasets/edit-01-edit.png"
  alt="Edit Dataset 폼"
  status="missing"
/>

## Merge

여러 데이터셋을 하나로 합칩니다. 같은 작업의 다른 시점/사람의 시연을 묶을 때 유용합니다.

<DocsImage
  src="/docs/datasets/edit-02-merge.png"
  alt="Merge Dataset 폼"
  status="missing"
/>

## Sensor / Robot 매핑 재지정

워크스페이스의 sensor/robot 구성이 바뀌면 기존 데이터셋이 호환되지 않습니다. Edit 폼 안의 매핑 셀렉트로 옛 이름을 새 이름에 다시 연결할 수 있습니다.
