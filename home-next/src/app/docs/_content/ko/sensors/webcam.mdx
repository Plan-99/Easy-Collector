UVC를 지원하는 일반 USB 웹캠은 모두 사용할 수 있습니다. Webcam Publisher 모듈에는 다음 사양이 기본 프리셋으로 들어 있고, Sensor Type 셀렉트에서 회사명을 골라 곧바로 등록할 수 있습니다.

| 회사 | 기본 해상도 | 비고 |
|------|------------|------|
| Logitech | 600 × 480 | 표준 UVC 웹캠 |
| APC | 640 × 480 | 산업용/저조도 UVC 웹캠 |

회사별로 해상도만 다르고 등록·동작 절차는 같습니다.

## 사전 조건

- [모듈 추가](/docs/modules)에서 **Webcam Publisher** 모듈 설치
- 웹캠 USB 연결 후 `ls /dev/video*`로 디바이스 인덱스 확인

## 등록 절차

<Steps>
1. **Sensors 페이지에서 Add Sensor를 누릅니다.**

2. **Sensor Type에서 webcam(Logitech / APC)을 선택하고 Device Index를 입력합니다.**

   `/dev/video0`이면 `0`, `/dev/video2`면 `2`입니다.

   웹캠 번호 확인:

   ```bash
   v4l2-ctl --list-devices
   ```

   디바이스마다 모델명과 USB 경로가 함께 표시되어 캡처카드/가상 V4L2 디바이스와 쉽게 구분됩니다. `v4l-utils` 패키지가 없으면 USB 심볼릭 링크로 대체할 수 있습니다.

   ```bash
   ls /dev/v4l/by-id/
   ```

   <DocsImage
     src="/docs/sensors/webcam-01-form.png"
     alt="SensorForm — Type=webcam, Device Index"
     status="missing"
   />

3. **저장 후 토글을 켜고 라이브 프리뷰를 확인합니다.**

   <DocsImage
     src="/docs/sensors/webcam-02-preview.png"
     alt="Webcam 라이브 프리뷰"
     status="missing"
   />
</Steps>

<Callout type="tip">
같은 PC에 같은 모델의 웹캠 여러 대를 꽂으면 인덱스가 부팅마다 바뀔 수 있습니다. udev 규칙으로 고정 심볼릭 링크를 만들어 두는 것을 권장합니다.
</Callout>
