이미 ROS 2 토픽으로 이미지를 발행하는 센서가 있다면 별도 모듈 없이 바로 등록할 수 있습니다.

## 사전 조건

- 외부 노드가 `sensor_msgs/Image` 또는 `sensor_msgs/CompressedImage` 토픽을 발행 중
- 토픽 이름과 메시지 타입을 정확히 알고 있음

## 등록 절차

<Steps>
1. **Sensors 페이지에서 Add Sensor를 누릅니다.**

2. **Sensor Type에서 custom을 선택합니다.**

3. **Read Topic, Read Topic Message Type, Resolution Width/Height를 입력합니다.**

   - **Read Topic** — 예: `/my_sensor/image_raw`
   - **Read Topic Message Type** — `sensor_msgs/CompressedImage` 또는 `sensor_msgs/Image`
   - **Resolution** — 발행되는 이미지 크기 (기본 640×480)

   <DocsImage
     src="/docs/sensors/custom-01-form.png"
     alt="Custom SensorForm — Read Topic / Msg Type / Resolution"
     status="missing"
   />

4. **저장 후 카드의 상태를 확인합니다.**

   custom 센서는 ON/OFF 토글 대신 **TOPIC ON / TOPIC OFF**로 표시됩니다. 외부 노드가 발행 중이면 자동으로 TOPIC ON이 됩니다.

   <DocsImage
     src="/docs/sensors/custom-02-topic-on.png"
     alt="Custom 센서 TOPIC ON 상태"
     status="missing"
   />
</Steps>

<Callout type="info">
Easy Trainer가 토픽을 직접 켜고 끌 수 없으므로 외부 노드를 따로 띄워야 합니다. 운영자 UI는 토픽이 발행되고 있는지만 감지합니다.
</Callout>
