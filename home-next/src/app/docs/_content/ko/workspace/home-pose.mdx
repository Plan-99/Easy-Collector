Home Pose는 **모든 에피소드의 시작 자세**입니다. 일관된 시작 자세는 학습 데이터의 분산을 줄여 정책 수렴을 빠르게 합니다.

## 설정 절차

<Steps>
1. **setting 탭에서 로봇을 클릭해 우측 config 패널을 엽니다.**

   <DocsImage
     src="/docs/workspace/home-01-section.png"
     alt="Home Pose 섹션 전체"
     status="missing"
   />

2. **RobotPendant 등으로 원하는 시작 자세를 만듭니다.**

3. **sync 버튼을 누르면 현재 관절값이 입력 칸에 자동 채워집니다.**

   <DocsImage
     src="/docs/workspace/home-02-sync.png"
     alt="sync 버튼"
     status="missing"
   />

4. **save 버튼으로 워크스페이스에 저장합니다.**

   <DocsImage
     src="/docs/workspace/home-03-save.png"
     alt="save 버튼"
     status="missing"
   />
</Steps>

<Callout type="tip">
Home Pose는 **물체와 충돌하지 않는 안전 자세**여야 합니다. 매 에피소드마다 이 자세에서 출발한다는 점을 고려하세요.
</Callout>
