워크스페이스에 추가된 센서만 데이터 수집 시 기록됩니다. 일반적으로 작업 영역을 보는 카메라 1–3대 정도를 추가합니다.

<Steps>
1. **setting 탭 → Sensor Setting expansion에서 + Add Sensor를 누르고 등록할 센서를 모두 선택합니다.**
