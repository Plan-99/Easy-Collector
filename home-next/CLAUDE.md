# Easy Trainer Homepage

## Overview
Easy Trainer의 공식 웹사이트. 제품 소개, 설치, 라이선스 관리를 위한 랜딩 페이지.

## 페이지 구조 (Single Page, 스크롤 방식)

### Section 1: Hero / 제품 소개
- Easy Trainer 소개 영상 (YouTube embed)
- 간단한 제품 설명 및 CTA

### Section 2: 요금제 플랜
- **Free** — 0원
- **Business** — 별도 컨택 (가격 비공개)

### Section 3: 고객사 후기
- 고객사 리뷰/testimonial 섹션

## 인증 및 라이선스 시스템

### 로그인
- Google OAuth를 통한 로그인/회원가입

### 설치
- Easy Trainer `.deb` 파일 다운로드 링크 제공
- Documentation 링크 제공

### License Key / 라이선스
- 가입한 이메일로 license key 발송
- License key는 **1대의 PC에만 바인딩** (하드웨어 잠금)
- License key 기반으로 요금제 플랜 적용
  - Free 플랜: 기본 기능
  - Business 플랜: 별도 컨택 후 적용

## 기술 스택
- 미정 (추후 결정)

## 참고
- 이 폴더는 Easy-Collector 본체와 별개의 웹사이트 프로젝트
- 본체(`src/`)의 UI(Vue/Quasar)와는 독립적으로 개발
